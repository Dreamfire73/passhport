Welcome to the PaSSHport administration panel

WARNING: This interface must be limited to passhport admin only.

This interface allows you to control all your SSH access, create users, targets (any ssh servers), organize groups... If you need any informations about passhport, you can find them on the online documentation.
PaSSHport is a free software published under GPLv3. You can find the sources and contribute directly on the gitub page.
